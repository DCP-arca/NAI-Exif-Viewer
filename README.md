# 개요
NAI로 생성된 이미지의 EXIF를 불러와 확인할 수 있는 간단한 어플리케이션입니다.

# 사용법
드래그드랍, 또는 빈공간을 클릭하여 png파일을 불러오세요.

각각 프롬프트, 네거티브 프롬프트, 생성 옵션, 기타 정보 순으로 EXIF 정보를 불러옵니다.
